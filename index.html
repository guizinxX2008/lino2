Jornada ao Tesouro Perdido no Egito

Passo 1: Início da Jornada

Você está no Cairo, Egito. Descobriu um antigo pergaminho que menciona um tesouro perdido em uma tumba antiga.

Passo 2: Escolha da Rota

O pergaminho revela dois caminhos: um pelo deserto e outro pelas catacumbas.

Passo 3: Desafios do Deserto

No deserto, você encontra uma antiga caravana. Os moradores oferecem duas opções: uma rota segura ou um atalho perigoso.

Passo 3: Catacumbas

Nas catacumbas, você encontra uma entrada para uma tumba com duas opções: um corredor apertado ou uma entrada oculta.

Passo 4: Rota Segura

A rota segura leva você a uma pirâmide antiga. Dentro da pirâmide, há dois caminhos: um para uma câmara secreta e outro para uma sala principal.

Passo 4: Atalho Perigoso

O atalho perigoso leva você a um oásis onde encontra um mapa antigo. O mapa revela o local da tumba.

Passo 4: Corredor Apertado

O corredor apertado leva você a uma sala com um enigma. Resolver o enigma revela a entrada para a tumba.

Passo 4: Entrada Oculta

A entrada oculta leva você a um salão com hieróglifos antigos. Você precisa decifrar os hieróglifos para encontrar o tesouro.

Passo 5: Câmara Secreta

Na câmara secreta, você encontra três sarcófagos: um dourado, um prateado e um de bronze.

Passo 5: Sala Principal

Na sala principal, você encontra uma caixa com um enigma que revela a localização do tesouro.

Passo 5: Mapa do Oásis

Seguindo o mapa, você encontra uma tumba antiga. Dentro, há uma pista final para o tesouro.

Passo 5: Enigma do Corredor

Resolver o enigma revela uma entrada secreta para o tesouro.

Passo 5: Hieróglifos

Decifrar os hieróglifos revela um mapa para uma câmara oculta.

Passo 6: Sarcófago Dourado

O sarcófago dourado leva a uma sala cheia de tesouros, mas o verdadeiro tesouro está escondido atrás de uma parede secreta.

Passo 6: Sarcófago Prateado

O sarcófago prateado contém um enigma que, se resolvido, revela o caminho para o tesouro.

Passo 6: Sarcófago de Bronze

O sarcófago de bronze contém um mapa que leva à localização do tesouro.

Passo 6: Resolver Enigma da Caixa

Resolver o enigma revela a localização do tesouro na tumba.

Passo 6: Entrar na Tumba

Dentro da tumba, você encontra um caminho oculto que leva ao tesouro.

Passo 6: Seguir a Entrada Secreta

A entrada secreta leva a uma sala onde o tesouro está escondido.

Passo 6: Seguir o Mapa

Seguindo o mapa, você encontra o tesouro escondido na câmara oculta.

Parabéns!

Você encontrou o tesouro perdido no Egito! Uma antiga riqueza de faraós, cheia de ouro e joias.
